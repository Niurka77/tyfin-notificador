# 🔔 Tyfin Notificador

Microservicio robusto diseñado para gestionar notificaciones push y alertas en tiempo real. Este proyecto demuestra la implementación de arquitecturas desacopladas y comunicación eficiente entre servidor y cliente.

### 🛠️ Tecnologías
- **Backend:** Node.js & Express
- **Real-time:** Socket.io
- **Notificaciones:** Firebase Cloud Messaging (FCM)

### 🌟 Características
- Envío de notificaciones push personalizadas.
- Gestión de eventos en tiempo real mediante WebSockets.
- Integración fluida con servicios de terceros.
